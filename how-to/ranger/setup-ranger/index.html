    Setting Up Ranger File Manager

    About Ranger

    Ranger is a console file manager with VI key bindings, featuring preview images, icons, and extensive customization options.

    Installation

    # Core packages
sudo pacman -S ranger atool ffmpegthumbnailer highlight mediainfo poppler ueberzug

# Nerd Fonts (choose one)
sudo pacman -S ttf-nerd-fonts-symbols
# OR for a full font:
sudo pacman -S ttf-firacode-nerd

    # Core packages
sudo apt install ranger atool ffmpegthumbnailer highlight mediainfo poppler-utils python3-ueberzug

# Nerd Fonts
wget -P /tmp https://github.com/ryanoasis/nerd-fonts/releases/download/v3.0.2/FiraCode.zip
sudo unzip /tmp/FiraCode.zip -d /usr/share/fonts/FiraCode
fc-cache -fv

    # Core packages
sudo dnf install ranger atool ffmpegthumbnailer highlight mediainfo poppler python3-ueberzug

# Nerd Fonts
sudo dnf install fira-code-fonts

    Configuration Setup

    1. Install the devicons plugin: 

      git clone https://github.com/alexanderjeurissen/ranger_devicons ~/.config/ranger/plugins/ranger_devicons

    2. Generate default configuration: 

      ranger --copy-config=all

    3. Enable features in ~/.config/ranger/rc.conf: 

      # Enable devicons
default_linemode devicons

# Enable image previews
set preview_images true
set preview_images_method ueberzug

    Preview Configuration

    • For kitty terminal: use set preview_images_method kitty
    • For terminology: use set preview_images_method terminology
    • For w3m: use set preview_images_method w3m

    Customization

    Popular Settings

    Add these to your rc.conf for enhanced functionality: 

    # Show hidden files by default
set show_hidden true

# Use one column mode when small window
set column_ratios 1,1

# Enable version control info
set vcs_aware true

    Troubleshooting

    Common Issues

    • No icons showing: Ensure Nerd Fonts are properly installed and terminal uses them
    • Preview not working: Verify ueberzug installation and terminal compatibility
    • Permission errors: Check file permissions in ~/.config/ranger/
