    Install Essential Software using Chocolatey

    To install all the essential software listed, run the following command in an elevated PowerShell (Run as Administrator):

    choco install hyper firefox obsidian flow-launcher everything vscode caffeine etcher git vim mremoteng powershell-core docker-desktop telegram python311 bitwarden -y

    Software Overview

    Hyper Terminal

    A sleek, extensible terminal built on Electron that leverages web technologies for customizability and performance. Supports plugins and themes to fit any workflow.

    Firefox

    A privacy-focused, open-source web browser known for speed, customizability, and support for modern web standards. Ideal for users who value security and freedom.

    Obsidian

    A powerful markdown-based knowledge management tool designed for connected thinking. Great for creating and maintaining personal wikis and notes.

    Flow Launcher

    A customizable, open-source productivity tool and application launcher for Windows. It supports plugins and quick file or web searches.

    Everything

    A lightning-fast file search engine for Windows that provides instant results. Perfect for finding files by name or extension across large systems.

    VS Code

    A lightweight, versatile code editor with extensive language support and a rich ecosystem of extensions. Built by Microsoft, it's ideal for developers of all skill levels.

    Caffeine

    A simple utility that prevents your computer from sleeping or activating a screensaver. Perfect for uninterrupted presentations or long-running tasks.

    balena Etcher

    A cross-platform tool for flashing OS images to SD cards and USB drives safely and easily. Streamlines the process for bootable media creation.

    Git

    A distributed version control system designed to handle projects of any size. Essential for software development and collaboration.

    Vim

    A highly efficient, keyboard-driven text editor with robust features and extensive customizability. Loved by power users and developers.

    mRemoteNG

    An open-source, multi-protocol remote connection manager. Supports RDP, SSH, and VNC, simplifying access to remote systems.

    PowerShell Core

    A cross-platform automation and configuration management tool built on .NET. Ideal for scripting, task automation, and system management.

    Docker

    A platform for developing, shipping, and running applications in lightweight containers. Simplifies deployment and environment consistency.

    Telegram

    A fast, secure messaging app focused on privacy and features like file sharing and group chats. Available across multiple platforms.

    Python3

    A versatile, high-level programming language known for its readability and broad library support. Widely used in web development, data analysis, artificial intelligence, and more.
