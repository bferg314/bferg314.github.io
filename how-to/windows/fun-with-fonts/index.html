    Fun with Fonts

    Installing New Fonts

    Step 1: Download Fonts

    Visit the Nerd Fonts website to download a variety of fonts designed for developers. Another good site for programmer fonts is Programming Fonts.

    Step 2: Install Fonts

    After downloading the font files (usually in .zip format), extract them and install the fonts by following these steps:

    1. Open PowerShell as an administrator.
    2. Navigate to the directory where the font files are located.
    3. Run the following command to install the fonts:
    echo "Install fonts"
$fonts = (New-Object -ComObject Shell.Application).Namespace(0x14)
foreach ($file in gci *.ttf)
{
    $fileName = $file.Name
    if (-not(Test-Path -Path "C:\Windows\fonts\$fileName" )) {
        echo $fileName
        dir $file | %{ $fonts.CopyHere($_.fullname) }
    }
}
cp *.ttf c:\windows\fonts\

    Note

    Ensure you have administrative privileges when running the PowerShell commands.

    Alternatively, you can install fonts by simply double-clicking the .ttf file and clicking the "Install" button.

    Viewing Installed Fonts

    Option 1: PowerShell

    [void] [System.Reflection.Assembly]::LoadWithPartialName("System.Drawing")
(New-Object System.Drawing.Text.InstalledFontCollection).Families

    Option 2: Windows Fonts Settings

    To view the installed fonts, open the Windows Fonts settings:

    1. Press Win + R to open the Run dialog.
    2. Type control fonts or fonts and press Enter.

    Warning

    After installing new fonts, you may need to restart your terminal emulator or code editor for the changes to take effect.

    Updating VS Code to Use the New Font

    1. Open VS Code.
    2. Go to File > Preferences > Settings.
    3. Search for Font Family.
    4. Add the name of your new font to the Editor: Font Family setting. For example:
    "editor.fontFamily": "Fira Code, Consolas, 'Courier New', monospace"

    Tip

    You can list multiple fonts as fallbacks in case the primary font is not available.

    Updating Hyper Terminal to Use the New Font

    1. Open the Hyper Terminal configuration file by running the following command in Hyper:
    hyper config
    1. In the configuration file, find the fontFamily setting and update it with the name of your new font. For example:
    module.exports = {
  config: {
    // ...existing code...
    fontFamily: '"Fira Code", "Consolas", "Courier New", monospace',
    // ...existing code...
  },
  // ...existing code...
};

    Additional Resources

    For more information on customizing your terminal and code editor, refer to the official documentation for VS Code and Hyper Terminal.

    Conclusion

    Installing and customizing fonts can greatly enhance your development experience. By following this guide, you can easily install new fonts and configure your favorite tools to use them. Enjoy your new, stylish coding environment!
